Felegram
The only reason I decided to start Felgram is the filtering that has been imposed on us Iranians. Things have gotten so bad that we can no longer use a simple app for our everyday needs.

Felegram — as the name suggests — is similar to Telegram. In this filtering situation, you can run it on a server and use it comfortably.

I built it with Next.js, Prisma, and PostgreSQL. The whole project is Dockerized to make things easy.

It's still not finished, but it's taking shape. I'd be happy if you check it out, share your thoughts, or even help build it.

How to run it:

1.Install Docker and Docker Compose

2.Clone the repository

3.Run this command in the terminal: docker-compose up --build

4.Open your browser at: http://localhost:3000

Everything spins up together. No manual setup, no headaches.
